# Swipe LP 定義・サイズ仕様

## 1. 定義（核）

- **1枚=1画面**: 1スライドがビューポートと同サイズで表示されること
- **ページスクロールなし**: スライド間の遷移はスワイプで行い、ページ自体はスクロールしないこと
- **フルブリード画像**: 画像は余白なしで画面全面に表示すること
- **CTA/進捗の常時表示**: CTA・進捗は画面内に固定表示（安全領域を考慮）
- **PC時の見せ方**: 中央430px幅のコンテナ + 背景ブラー

## 2. 画面サイズ基準

- **縦比率**: 9:16 を基準
- **推奨解像度**: 1080x1920 以上（2x推奨）
- **安全領域**: 上下48-64pxはテキスト・重要要素を避ける

## 3. 表示ルール

- **スワイプ方向**: 縦方向（上下）
- **画像表示**: `object-fit: cover` を基本（余白を出さない）
- **CTA位置**: 画面下部固定（safe-area-inset-bottom考慮）
- **進捗バー**: 画面上部固定

## 4. PC表示

- **コンテナ幅**: max-width 430px
- **背景**: スライド画像のブラー + グラデーションレイヤー
- **CTA/進捗**: コンテナ内に収める

## 5. 例外/補足

- 画像が規格外の場合は、トリミング前提で `object-fit: cover` に統一
- スクロール型ビューアは補助モードとし、スワイプ型を標準とする
